Introduction to SSM

SSM consists of three frameworks: Spring, SpringMVC, and MyBatis. As a Bean container, Spring is a lightweight Control Inversion (IOC) and Aspect Oriented (AOP) container framework. SpringMVC separates the roles of controllers, model objects, dispatchers, and handler objects. MyBatis uses simple XML or annotations for configuration and raw mapping, mapping interfaces and Java POJOs to records in the database.
